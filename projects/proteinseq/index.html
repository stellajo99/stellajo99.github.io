This study investigates how well different deep learning models can classify protein sequences based only on amino acid embeddings.
Protein sequences are vital in bioinformatics, but their sheer volume demands automated, accurate classification to understand function and taxonomy efficiently.

Inspired by the sequential nature of amino acids (time-series-like), this project compares RNN, LSTM, and Transformer architectures to evaluate which is more effective for protein classification.

    • Long Short-Term Memory (LSTM)
    • Transformer (Attention-based)

  • Identify which architecture is most suitable for order-sensitive protein sequence data.

    • Caenorhabditis elegans, Homo sapiens, Mus musculus.

Preprocessing:

  • Labeled for supervised learning.
  • Split into train:validation:test sets (60% : 20% : 20%).
  • Implemented in Google Colab GPU environment.

Models:

  • All models share the same hyperparameters: hidden layers, optimizer, loss function, epochs, batch size.
  • Early stopping applied if validation loss does not decrease for 10 epochs.

  • Transformer: Uses Attention mechanisms to process input in parallel rather than sequentially, theoretically faster but may underperform on biological sequences.
