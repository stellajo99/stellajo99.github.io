A Study on Personalized Travel Route Recommendation Using LSTM

August 7, 2022 ¬∑ 3 min

üìå Overview#

This research aims to design a personalized travel route recommendation system that goes beyond simple location lists by also predicting how long to stay at each place.
Unlike prior studies that often rely on external preference data, this work uses only raw GPS trajectory data, clustering similar travel patterns and applying an LSTM Encoder-Decoder to generate time-aware travel sequences.
The goal is to make travel planning more efficient and user-centric.

üéØ Research Objectives#

  • Analyze GPS trajectory data to identify individual travel patterns
  • Convert travel paths into string sequences for similarity comparison
  • Cluster users based on sequence similarity (edit distance)
  • Use Relaxed LCS to assign users to the most relevant cluster
  • Predict personalized, time-included travel sequences using LSTM
  • Compare prediction accuracy with baseline methods

üìö Methodology#

Dataset:

  • Microsoft Asia GeoLife GPS Trajectories (182 users, ~15,000 routes, Beijing)

Preprocessing:

  • Stay Point Detection using scikit-mobility
  • Reverse Geocoding via Google Places API
  • Transform stay points into unique character sequences representing locations and durations

Clustering:

  • Hierarchical clustering with Cosine similarity and Levenshtein distance
  • Relaxed Longest Common Subsequence (LCS) for user-to-cluster assignment

Modeling:

  • Sequence-to-Sequence LSTM Encoder-Decoder architecture
  • Sliding window data generation for training/validation/testing
  • Performance evaluated with MA (location-only accuracy) and MRE (order-sensitive accuracy)

üìä Key Findings#

  • Clustering Results:
    Hierarchical clustering using Levenshtein distance showed more stable clusters (average silhouette score ‚âà 0.10) compared to other clustering methods.

  • Model Performance:

    • Average MA (location prediction) > 0.5 across clusters
    • Average MRE (sequence order accuracy) ‚âà 0.5, which is promising given the complexity
    • The proposed method achieved 2x higher sequence accuracy than baseline approaches (~50% vs. 19%)
    • Training Loss: Cluster-specific loss values remained low overall (e.g., most clusters achieved final loss between ~0.14 and ~0.28), indicating good convergence and minimal overfitting when validated on unseen data

üí° Insights & Discussion#

  • Contribution:
    This study shows that meaningful, personalized travel recommendations can be generated using only GPS trajectory data, minimizing dependency on external preference information.

  • Limitations:

    • The dataset used was not collected exclusively for travel; some recommended stops may not align with typical travel behavior.
    • The model currently does not fully learn real-world distance constraints between locations.

  • Future Work:

    • Apply the method to real tourist data for higher travel relevance.
    • Integrate spatial distance constraints into the prediction layer.
    • Extend this foundation into a practical travel planning service.
