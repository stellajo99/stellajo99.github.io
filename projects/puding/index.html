Our Project PUDING is a mobile coding companion that helps users set coding goals, solve problems, and earn rewards to keep a virtual pet happy. The app guides users through goal setting, problem solving, and screenshot submission for motivation, and requires login to access personalized tasks and progress tracking!

üîß Key Features#

  • Simple Task: Set personal coding goals with custom time, repeat frequency, and problem details.
  • Moderate Task: Solve recommended coding problems and submit proof of completion.
  • Complex Task: Submit screenshots of solved problems to earn coins and keep the virtual puppy happy, with dynamic emotion status bars.

  • Gamification: Virtual pet status system with rewards

üìë How It Works#

  1. Goal Setting:
    Users navigate to the ‘Work’ tab to add and manage daily coding goals.
    They can set the time, repetition frequency, and the specific problems they wish to solve.

  2. Problem Solving:
    Users access daily coding problems, solve them externally, then return to upload a screenshot as confirmation.
    Each problem has a unique ID and link for easy access.

  3. Reward & Pet Interaction:
    When a user submits proof of solving a problem, they earn coins which can be spent in the store to buy food and toys for a virtual puppy.
    The pet‚Äôs happiness and hunger bars update dynamically, visually showing the impact of consistent coding practice.
