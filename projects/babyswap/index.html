BabySwap is a web application that helps parents find reliable and affordable babysitting by swapping babysitting hours within a trusted community.
Families earn and spend points instead of cash, making childcare more flexible and cost-effective.

This project was delivered as part of SIT725 ‚Äì Applied Software Engineering, where our team built BabySwap from initial requirements to final deployment.

üîß Key Features#

  • User Registration & Login: Secure authentication and session handling
  • Babysitting Posts: Create, view, accept, and manage babysitting offers and requests
  • Points System: Earn points by babysitting for others, spend points when requesting help
  • Account Management: View/update account info, check points balance, delete account
  • Notifications: Real-time updates for post status changes
  • Testing: Comprehensive unit tests (Jest) and end-to-end tests (Playwright)
  • Unit & E2E Testing: Robust test coverage using Jest and Playwright
  • Containerization: Fully containerized with Docker for consistent development and deployment
